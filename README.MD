# CRM CRUD
Proyecto CRUD diseñado para gestionar distintos mensajes de contacto de diferentes clientes para fines de evaluacion y seguimiento, incluyendo numero de telefono y mensaje y el nombre del cliente.

## Requisitos

- .NET 8 SDK

## Ejecución

**API:**
```bash
cd CRMCrud.API
dotnet run
```
Disponible en `http://localhost:5169`

**Frontend:**
```bash
cd frontend
index.html 
```
Abriria el crud de forma automatica en la ruta deseada
